# 🚀 AIO VPN Unlocker

**AIO VPN Unlocker** is a Telegram-based VPN config unlocker bot that allows users to decrypt and view configurations from multiple VPN tunneling apps.

Simply send a supported file or link to the bot and get instant unlocked output.

## 📂 Supported Formats

Send a file with one of these extensions or prefixes:

### 🔹 Core Formats

- EHI: .ehi, .ehil
- HC: .hc
- NPVT: .npvt
- DARK: .dark (darktunnel://)
- EV2RAY: .v2
- SIP: .sip
- HAT: .hat
- NETMOD: .nm (nm-type://)
- OPENTUNNEL: .tnl
- SLIPNET: .slipnet (slipnet://, slipnet-enc://)
- TLS: .tls
- LINKLAYER: .lnk
- MINA: .mina
- SSC: .ssc (ssc://)
- More...
